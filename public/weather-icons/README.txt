Place PNG icons for each weather condition here. Recommended icon names:

- sunny-[theme].png
- cloudy-[theme].png
- rain-[theme].png
- snow-[theme].png
- thunder-[theme].png
- fog-[theme].png
- partlycloudy-[theme].png
- drizzle-[theme].png
- wind-[theme].png

Where [theme] is one of: default, dark, green, navy, dark-green, yellow, light-blue, red, purple, night, ambient

Example: sunny-default.png, rain-dark.png

Icons should be 80x80px PNGs, transparent background, colored to match the theme.
